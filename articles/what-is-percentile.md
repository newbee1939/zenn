## パーセンタイル(percentile)とは

**データを小さい順に並べたとき、初めから数えて全体の100α%に位置する値**を**100αパーセンタイル**と言います。（0 <= α <= 1）

## Google Cloudのパーセンタイル（レイテンシの例）

参考: [レイテンシの指標](https://cloud.google.com/spanner/docs/latency?hl=ja)

```
50 パーセンタイルのレイテンシ

最も処理時間の短いリクエストの 50% の最大レイテンシ（秒）。たとえば、50 パーセンタイルのレイテンシが 0.5 秒の場合、Spanner は、50% のリクエストを 0.5 秒未満で処理しています
```

```
99 パーセンタイルのレイテンシ

最も処理時間の短いリクエストの 99% の最大レイテンシ（秒）。たとえば、99 パーセンタイルのレイテンシが 2 秒の場合、Spanner は、99% のリクエストを 2 秒未満で処理しています。
```

e.g. 99パーセンタイルのレイテンシが80msの場合、リクエスト全体の99%は80ms以下でデータを返せていることになる
